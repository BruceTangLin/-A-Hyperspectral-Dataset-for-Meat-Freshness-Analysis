# -A-Hyperspectral-Dataset-for-Meat-Freshness-Analysis
本项目同时开源了一个用于肉类新鲜度分析的近红外高光谱数据集。数据集由波段范围为 900–1700 nm 的近红外高光谱相机采集。数据在连续 三天内进行采集，共划分为 19 个不同时段，用于刻画肉类新鲜度随时间变化的过程。其中选取皮带背景，最新鲜时刻的瘦肉和肥肉，以及最不新鲜时刻的瘦肉和肥肉作为端元光谱。并基于线性混合模型（LMM）与全约束最小二乘（FCLS）方法对高光谱图像进行解混，得到每个像素在上述端元上的丰度估计图。最终，通过端元丰度的空间分布及其随时间变化的统计特征，实现对肉类新鲜度状态的定量判定与可视化表达。
